==================
Texts from element
==================

The element texts section provides all text properties from the `text field`_ and `dynamic text`_ integrated in the `element`_. The text properties available are:

* Text
* Source of the text
* Size
* Font
* Color
* Frame
* Width
* Position
* Rotation
* Alignment
