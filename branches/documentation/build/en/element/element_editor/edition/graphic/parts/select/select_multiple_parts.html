Select multiple parts¶

As many CAD tools, many parts (Line, Rectangle, Ellipse, Arc, Polygon, Terminal, text and dynamic text field) from QElectroTech workspace can be selected at the same time.

Select multiple objects using keyboard and mouse¶

QElectroTech allows selecting multiple parts from the workspace, graphical area, combining the use of the keyboard and mouse.

  1. Select the first part.

  2. Press Ctrl.

  3. Select the rest of parts without releasing Ctrl.

    Figure: QElectroTech schema

Select multiple objects by selecting area¶

QElectroTech allows selecting all parts from an area using the mouse to select the desired area.

  1. Left click on the initial point from the rectangular area to be selected and displace the mouse without releasing the button.
