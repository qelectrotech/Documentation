Clean projectÂ¶

QElectroTech stores on the project database every title block, conductor, element, etc. which is introduced by the user, if the user deletes one element or replace the title block from the folio, the element or title block will be deleted from the folio, but, it will still be storage at the project database.

QElectroTech allows cleaning the project database from Menu bar.

  1. Select Project > Clean project menu item to open the cleaning project PopUP window.

Figure: QElectroTech cleaning project PopUP window

  1. Select the check buttons desired (templates, elements, categories).
  2. Press the button OK to clean the project and close the cleaning project PopUP window.

Note

Cleaning the project is recomended to reduce the size of the project file and inclease the loading speed.
