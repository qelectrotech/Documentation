Start QElectroTech on WindowsÂ¶

After installation, QElectroTech can be launched from the Windows Start Menu. If the shortcut icon has been created at the desktop during installation, QElectroTech can also be started from that icon.

  1. Click the Windows [Start] button.
  2. Select All Programs.
  3. Open QElectroTech program group.
  4. Click QElectroTech icon.
