After installation, QElectroTech can be launched from the Windows Start Menu. If the shortcut icon has been created on the desktop during the installation, QElectroTech can also be started from that icon.

  1. Click the Windows [Start] button.
  2. Select All Programs.
  3. Open the QElectroTech program group.
  4. Click the icon from QElectroTech.
