QElectroTech defines the working area from the workspace as a grid of columns and rows. All columns have the same width and all the rows have the same height. Bellow, a folio with 17 columns (0 to 16) and 8 rows (A to H) is shown.

Figure: QElectroTech folio example

At the dimensions of folio section from the folio properties panel, the following parameters can be defined:

  1. Number of columns.
  2. Width from columns.
  3. Column headers display or hidden.
  4. Number of rows.
  5. Height from rows.
  6. Row headers display or hidden.

Figure: Folio dimensions properties

The version 0.7 from QElectroTech works with pixels and there is no pre-defined folio sizes. The table bellow defines the pixels dimensions according ISO 216.
