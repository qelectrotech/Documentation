Panels¶

Cell properties panel¶

The Cell properties panel displays the properties from the selecte cell.

Figure: QElectroTech Cell properties panel

To display the Cell properties panel:

  1. Select Settings > display > Cell properties menu item to display the Cell properties panel.

Undo panel¶

The Undo panel displays the history since the last time that the Title Block was saved. Once the Title Block is saved, undo panel is automaticall cleaned.

Figure: QElectroTech Undo panel

To display the Cell properties panel:

  1. Select Settings > display > Undo menu item to display the Undo panel.
