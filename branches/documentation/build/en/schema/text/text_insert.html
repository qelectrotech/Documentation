Note

To insert object to the workspace more easily, the folio grid can be displayed from Display > Display the grid or from the toolbar icon .

The text field can only be added to the workspace by the toolbar.

  1. Select the icon from the toolbar to add a rectangle.
  2. Click on the workspace place point where the text field should be placed.
  3. Write the desired text.
  4. Click anywhere from the workspace to top the edition process and save the text content.

Note

If the toolbar is not displayed, it can be displayed from Settings > Display > Add.
