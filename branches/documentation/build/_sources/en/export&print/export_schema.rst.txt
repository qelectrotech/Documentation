Export schema
=============

QElectroTech allows exporting the different `folios`_ to many type of formats: **PNG**, **JPEG**, **Bitmap**, 
**SVG** and **DWX**.   

    1. Select **File > Export** menu item to display export PopUP window.

        .. figure:: graphics/qet_file_menu.png
            :align: center

            Figure: QElectroTech File menu

    .. note::

        QElectroTech allows using keyboard shortcut to increase the working efficiency.

            1. Press ``Ctrl + Shift + X`` to display export PopUP window.

        For more information about QElectroTech keyboard shortcut, please refers to `Menu bar`_ section.
   
    2. Choose the `folios`_ you wish to export and specify their size.

        .. figure:: graphics/qet_file_export_window.png
            :align: center

            Figure: QElectroTech export PopUP window 
   
    3. Choose target directory and format.
    4. Choose rendering options.

        .. figure:: graphics/qet_export_formats.png
            :align: center

            Figure: QElectroTech exporting formats combo box
   
    5. Press **Export** button to create the files.

QElectroTech creates one file for each folio choosed at the target directory defined. 

.. seealso::

    QElectroTech allows predefining export settings for reducing the working configuration 
    effort each time that a `project`_ have to be exported. Please refers to `Export settings`_ 
    section for more information.
