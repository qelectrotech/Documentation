=============
Delete object
=============

Choosing the correct elements, connectors, tables, etc. from the beginning is always nice, even so, 
QElectroTech allows deleting all type of object that can be add to the folios (elements, connectors, 
text fields and pictures).

.. note::

   Multiples objects can be deleted at the same time, pressing ``Ctrl`` is necessary to select more 
   than one object.

Deleting objects can be done from the Menu bar, toolbar, by right click and from the corresponding 
keyboard shortcut.

Delete object from Menu bar
~~~~~~~~~~~~~~~~~~~~~~~~~~~

    1. Select the object which should be delete.
    2. Select **Edit > Delete** menu item to delete the object.

.. figure:: graphics/qet_menu_edit.png
   :align: center

   Figure: QElectroTech Edit menu

Delete object from toolbar
~~~~~~~~~~~~~~~~~~~~~~~~~~

    1. Select the object which should be deleted.
    2. Select the icon |icon_delete| from the toolbar to delete the object.

.. |icon_delete| image:: graphics/qet_delete_icon.png

.. note::

   If the toolbar is not displayed, it can be displayed from **Settings > Display > Tools**.

Delete object by right click
~~~~~~~~~~~~~~~~~~~~~~~~~~~~

    1. Right click on the object which should be deleted.
    2. Select the option **Delete** to delete the object.

.. figure:: graphics/qet_element_right_click.png
   :align: center

   Figure: QElectroTech right click PopUP window

Delete object using keyboard shortcut
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

QElectroTech allows using keyboard shortcut to increase the working efficiency.

    1. Select the object which should be deleted.
    2. Press ``delete`` to delete the object.

.. seealso::

    For more information about QElectroTech keyboard shortcut, please refers to `Menu bar <../../en/interface/menubar.html>`_ section.
