Interface elements
====================

QElectroTech has been designed using `Qt`_ framework and widget toolkit, this 
means that the main window from QElectroTech is the same for all platforms where is 
available (Windows, Linux/Unix and MacOS). The main window from QElectroTech looks 
like the figure bellow.

.. figure:: graphics/qet_gui.png
   :align: center

   Figure: QElectroTech main window

.. role:: bluetext

The main window from QElectroTech contains the following areas:

    1. :greentext:`Menu bar`
    2. :bluetext:`Toolbar`
    3. :orangetext:`Panels`
    4. :yellowtext:`Workspace`
    5. :redtext:`Projects tab`
    6. :lilactext:`Folios tab`
    7. :browntext:`Help bar`
    8. :pinktext:`Search Menu`

.. _Qt: https://www.qt.io/
