====================================
Define title block extra information
====================================

Define extra information from Menu bar
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

    1. Select **Edit > Edit extra information** menu item to display the `extra information PopUp window`_.

        .. figure:: graphics/qet_titleblockeditor_edit.png
            :align: center

            Figure: QElectroTech title block editor Edit Menu

    2. Introduce the desired information to the text field.
    3. Press the button **OK** to accept and close the `extra information PopUp window`_.

Define extra information using keyboard shortcut
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

QElectroTech allows using keyboard shortcut to increase the working efficiency.

    1. Press ``Ctrl + y`` to display the `extra information PopUp window`_.
    2. Introduce the desired information to the text field.
    3. Press the button **OK** to accept and close the `extra information PopUp window`_.

.. seealso::

    For more information about QElectroTech keyboard shortcut, please refers to `Menu bar`_ section.
