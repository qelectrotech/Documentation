Delete Folio
============

QElectroTech allows delete folions from the Menu bar and from the project panel.

Delete folio from Menu bar
~~~~~~~~~~~~~~~~~~~~~~~~~~~

    1. Select **Project > Delete this folio** menu item to delete the active folio.

.. figure:: graphics/qet_project_menu.png
   :align: center

   Figure: QElectroTech Project menu 

Delete folio from project panel
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

    1. Right click on the folio which should be deleted.
    2. Click the option **Delete this folio** to delete de folio selected.

.. figure:: graphics/qet_project_panel_delete.png
   :align: center

   Figure: QElectroTech Project panel 

.. note::

   If the project panel is not displayed, it can be displayed from **Settings > Display > Projects**
