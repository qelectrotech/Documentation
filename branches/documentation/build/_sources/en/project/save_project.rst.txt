.. _en/project/save_project

============
Save Project
============

The current `project`_ can be saved from the `Menu bar`_, `toolbar`_ and using the corresponding 
keyboard shortcut.

One `project`_ is conformed by only one file with the format ``.qet``. The ``.qet`` extension 
is the native extension from QElectroTech.

Save project from Menu bar
~~~~~~~~~~~~~~~~~~~~~~~~~~

    1. Select **File > Save** menu item to save the active `project`_.

.. figure:: graphics/qet_file.png
   :align: center

At the case that the `project`_ has already been saved and wants to be saved as a different `project`_, 
different name and/or directory:

    1. Select **File > Save as** menu item to save the active `project`_ as a new `project`_.

Save project from toolbar
~~~~~~~~~~~~~~~~~~~~~~~~~

    1. Select the icon |icon_save| from the `toolbar`_ to save the active `project`_.

.. |icon_save| image:: graphics/qet_toolbar_save.png

At the case that the `project`_ has already been saved and wants to be saved as a different `project`_, 
different name and/or directory:

    1. Select the icon |icon_save_as| from the `toolbar`_ to save the active `project`_ as a new `project`_.

.. |icon_save_as| image:: graphics/qet_toolbar_save_as.png

.. note::

   If the `toolbar`_ is not displayed, it can be displaed from **Settings > Display > Tools**

Save project using keyboard shortcut
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

QElectroTech allows using keyboard shortcut to increase the working efficiency.

    1. Press ``Ctrl + s`` to save the active `project`_.

.. seealso::

    For more information about QElectroTech keyboard shortcut, please refers to `Menu bar`_ section.

.. note::

   QElectroTech allows creating backup from the `project`_ periodically. This option should be 
   configurated at **Settings > Configure QElectroTech > Projects**.
